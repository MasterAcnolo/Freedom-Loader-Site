## Theme Workshop

The Theme Workshop allows you to create and manage custom themes for Freedom Loader.

### Creating a Theme

1. Open the theme editor at https://masteracnolo.github.io/Freedom&20Loader&20Workshop/create.html
2. Customize colors, fonts, and layout using the visual editor
3. Preview changes in real-time on the left panel
4. Click "Export .zip" to download your theme

### Installing a Theme

Extract the theme ZIP file and place it in your Freedom Loader themes folder:
`%APPDATA%\Freedom Loader\themes`

Then restart Freedom Loader and select your theme from Settings > Appearance. Or use Refresh Button next to theme selector.

### Submitting a Theme

Share your theme with the community:
1. Export your theme as ZIP
2. Create a GitHub issue on the [workshop repository](https://github.com/MasterAcnolo/Freedom-Loader-Workshop)
3. Include your theme ZIP and preview image
